📌 Project Description
This project analyzes global progress toward sustainable development using 2023 data. The objective is to provide a comparative report across various countries, focusing on economics, demographics, and CO₂ emissions, to support a non-profit organization in defining intervention priorities.

📊 Datasets Used
The data originates from three primary sources:

Global Country Information Dataset (2023): Economic and environmental indicators.

Global Data on Sustainable Energy: In-depth insights into the energy transition.

Global Missing Migrants Dataset: Data on migration flows and missing migrants.

🛠️ Technical Skills Developed
Data Cleaning: Handling null values and standardizing country names.

Comparative Analysis: Utilizing JOIN operations to merge economic data with energy data.

Advanced Aggregations: Calculating weighted averages, per capita emission percentages, and growth rates.

Ranking: Implementing Window Functions (e.g., RANK()) to classify the most sustainable countries.
